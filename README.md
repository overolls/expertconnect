# ExpertConnect - Consultant/Client Matching Platform

**Live Demo:** https://expertconnect1-bb330.web.app/

## Project Overview

ExpertConnect is a full-stack web application that connects consultants with clients. Built as my senior computer science project, I self-taught React, Express, and Stripe integration to complete it entirely on my own.

## Tech Stack

### Frontend
- React (self-taught from documentation and tutorials)
- Stripe Checkout / Payment Elements
- CSS3, HTML5
- Deployed on Firebase Hosting

### Backend
- Express.js (Node.js)
- Stripe API (payment integration in test mode)
- Firebase Firestore (NoSQL database)
- Firebase Authentication
- img.bb API (image hosting for consultant profiles)

### Tools & Environment
- Git & GitHub for version control
- VS Code
- Firebase CLI
- Stripe CLI (for local webhook testing)
